Instructions for Use of Code Editor

  • 1) Never refresh Code Editor Tab because after refreshing of webpage all content in that tab will be removed automatically.
  • 2) All New Template pages will open in new Tab, So don't remove the Tab in which Code Editor is open.
  • 3) You can use COPY option to copy the code which you have typed in coding area.
  • 4) The changes done in coding area will automatically update in output section ! No need of refreshing the page again & again.
  • 5) You can embed images by URL of thet image. NOTE : You cannot use images from your device in online code editor.
